# clear-read
A minimalist, browser-native reading utility that strips away web clutter and distractions through custom CSS injection. Focus on the content, not the noise.

## Why this exists
Modern websites are cluttered with ads, sidebars, and trackers. This tool uses simple CSS injection to strip a webpage down to its core text, formatted for maximum readability.

## Features
- **Client-Side Only:** No backend, no APIs, no tracking.
- **E-Ink Aesthetic:** Uses a high-readability serif font and a soft grayscale palette.
- **Distraction-Free:** Aggressively hides navbars and ads via CSS injection.

## How to use
1. Clone this repo.
2. Open `index.html` in any browser.
3. Paste a URL and hit "Render."
